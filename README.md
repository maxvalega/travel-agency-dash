# KRAFT YOUR TRIP - Travel Agency Dashboard

A simple tool to help travel agencies run their business.

## What it does

- **Bookings**: Manage trips and reservations.
- **Clients**: Store customer details and information.
- **Packages**: Create and track travel packages.
- **Reports**: See business data and statistics.
- **Settings**: Adjust dashboard options.

## How to use it

1. **Install packages**:
   ```bash
   npm install
   ```

2. **Start the app**:
   ```bash
   npm run dev
   ```

3. Open the link shown in the terminal (usually `http://localhost:5173`).
